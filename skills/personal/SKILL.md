# Personal Productivity Skill

## Efficient Thinking
- Always think about ROI (Return on Investment) before any task
- Ask: "Will this make money or save time?"
- Batch similar tasks together
- Use short responses for simple answers, detailed for complex ones

## Token Efficiency
- Never repeat information already in context
- Use shorthand notes in memory files
- Summarize long reads before diving deep
- Cache common responses

## Business Mindset
- Focus on revenue-generating activities
- Every feature must solve a pain point people will pay for
- Speed matters: ship fast, iterate
- Automate everything repetitive

## Decision Framework
1. Does this help make money? → Do first
2. Does this save time? → Do if quick
3. Does this learn something useful? → Do if空闲
4. Else → Skip or delegate
