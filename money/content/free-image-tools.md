# Free Image Tools for Content Creation

## For Graphics/Design

### Canva (Free Tier)
- **URL:** canva.com
- **Best for:** Social media templates, quotes, infographics
- **Free Features:**
  - 5,000+ templates
  - 1GB storage
  - Basic animations
  - Millions of free photos/icons
- **Pro Features ($$):** Brand kit, magic resize, premium elements

### Canva Alternative: Figma (Free)
- **URL:** figma.com
- **Best for:** Custom graphics, mockups
- **Free Features:** Unlimited files, basic collaboration

## For Photos

### Unsplash
- **URL:** unsplash.com
- **Best for:** High-quality stock photos
- **License:** Free, no attribution required
- **Search:** Natural, tech, office, lifestyle

### Pexels
- **URL:** pexels.com
- **Best for:** Stock videos + photos
- **License:** Free, no attribution

### Pixabay
- **URL:** pixabay.com
- **Best for:** Illustrations, vectors, videos
- **License:** Free, no attribution

## For AI Images (Free)

### Bing Image Creator (DALL-E 3)
- **URL:** bing.com/create
- **Best for:** Custom illustrations, unique images
- **Free:** 100 boosts/week (Microsoft account)
- **Quality:** Excellent

### Leonardo.ai
- **URL:** leonardo.ai
- **Best for:** Artistic, stylized images
- **Free:** 150 tokens/day
- **Great for:** Tech illustrations

### Ideogram
- **URL:** ideogram.ai
- **Best for:** Text-in-images (logos, quotes)
- **Free:** 100 images/day

## Quick Workflow

### For Quote Posts:
1. Open Canva → Search "quote"
2. Pick template → Add your text
3. Download PNG → Post

### For Blog/Tweet Images:
1. Canva → Social Media template
2. Add your stats/screenshot
3. Brand colors → Export

### For Custom Illustrations:
1. Bing Image Creator → Describe what you want
2. Download → Clean up in Canva
3. Add text overlay

---

## Pro Tips
- **Consistency wins:** Use same colors/fonts across posts
- **Text overlays:** Always add text so images work alone
- **Alt text:** Add descriptions for accessibility
- **Size matters:** 
  - Twitter: 1200x675 (16:9)
  - Instagram: 1080x1080 (1:1) or 1080x1350 (4:5)
